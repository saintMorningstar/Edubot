# Ambient Sound Files

The four `.wav` files in this folder are **silent placeholders**.  
Replace each one with real ambient audio to activate the background sound system.

## Themes

| File | Screen context | Suggested style |
|------|---------------|-----------------|
| `home.wav` | Dashboard / welcome | Soft synth pads, gentle wind chimes |
| `learning.wav` | Colors / Numbers / Shapes | Light nature ambience, slow piano |
| `coding.wav` | Coding / missions | Subtle futuristic robotic hum, soft arpeggios |
| `sleep.wav` | Nursery rhymes / story mode | Very slow piano, soft lullaby pads |

## Format requirements

- **Format:** MP3 (rename to `.wav` if your source is already WAV) or WAV
- **Duration:** 60–180 seconds — the service loops seamlessly
- **Sample rate:** 44 100 Hz or 22 050 Hz
- **Channels:** Mono or Stereo
- **Bit depth:** 16-bit (WAV) / 128 kbps (MP3)
- **Peak level:** Keep peaks below −12 dBFS so the mix stays gentle

## Good sources for free ambient audio

- freesound.org (Creative Commons licence)
- pixabay.com/music (royalty-free)
- soundsnap.com
- Generate with Audacity (built-in tone/chord generator + reverb)

## Looping tips

Trim the file so the beginning and end are at the same volume level.  
In Audacity: Effect → Fade In / Fade Out on the first and last 2 seconds.  
This prevents any click or volume jump when the loop restarts.
